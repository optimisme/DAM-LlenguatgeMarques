# Room 4: Build Semantic Structure

## Problem
The page preview uses `<div>` elements with class names that suggest what semantic element they should be.

## Task
Edit `index.html` to replace the div elements with their proper HTML5 semantic equivalents:
- `.site-header` → `<header>`
- `.site-nav` → `<nav>`
- `.site-main` → `<main>`
- `.blog-article` → `<article>`
- `.site-footer` → `<footer>`

## Success Condition
The `.page-preview` container must contain all five semantic elements: `<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`

## What You Can Edit
- `index.html` only

## Hint
Replace `<div class="site-header">` with `<header class="site-header">` or just `<header>`. Keep the class if you want, or remove it.

## Common Mistakes
- Only changing the class name instead of the tag itself
- Forgetting to change both opening AND closing tags
- Breaking the nesting structure
- Adding semantic elements instead of replacing the divs
