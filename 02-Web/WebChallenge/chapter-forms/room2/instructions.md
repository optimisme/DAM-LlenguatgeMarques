# Room 2: Use Correct Input Types

## Problem
All inputs are using `type="text"`, but HTML5 provides specialized input types for better UX and validation.

## Task
Edit `index.html` to change the input types:
- Email input: `type="email"`
- Phone input: `type="tel"`
- Age input: `type="number"`

## Success Condition
- `#user-email` must have `type="email"`
- `#user-phone` must have `type="tel"`
- `#user-age` must have `type="number"`

## What You Can Edit
- `index.html` only

## Hint
Change `type="text"` to the appropriate type for each field.

## Common Mistakes
- Using `type="phone"` instead of `type="tel"`
- Using `type="integer"` instead of `type="number"`
- Misspelling the type values
- Removing other attributes when changing type
