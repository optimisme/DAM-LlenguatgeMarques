# Room 3: Fix the Box Model

## Problem
The boxes are cramped together with no spacing between them and no internal padding.

## Task
Edit `styles.css` to add proper spacing to the `.box` class:
- Add `margin-bottom` of at least 10px (space between boxes)
- Add `padding` of at least 15px (space inside each box)

## Success Condition
- Each `.box` must have at least 10px margin-bottom
- Each `.box` must have at least 15px padding

## What You Can Edit
- `styles.css` only

## Hint
Change `margin-bottom: 0;` to `margin-bottom: 15px;` and `padding: 0;` to `padding: 20px;`

## Common Mistakes
- Confusing margin (outside spacing) with padding (inside spacing)
- Forgetting the unit (px, rem, em)
- Using margin-top instead of margin-bottom
- Adding margin to the container instead of the boxes
